Local Access Network

Local Access Network or LAN entails connecting computers locally on a single or small network of routers. This has the benefit of creating the fastest possible sharing speeds and preventing outside intrusions but is only applicable when devices are connected in close proximity. LAN is often used for single location business, households, and playing local games.

Virtual Networks

A Virtual Network functions under the exact same conditions as a LAN connection, but instead is performed wirelessly. A Virtual Private Network or VPN connects individual private networks over the public internet. It establishes a direct connection between devices which then enables users to send and receive data wirelessly. Users on computer “A” for instance, are able to see the contents on computer “B”. This is similar to how shared Google folders allow multiple people access to the same files on remote computers. This allows businesses, for example, that have multiple branches, to access the same records from different locations without having them publicly available.
