Browsers Routers Networks Network Addresses

Function

Most people associate routers with Wi-Fi, but it actually plays a much more integral part in sending and requesting data from devices connected to it. The basic function of a router is to request information from the internet and, in certain circumstances, allow other internet users to request information from devices connected to it. Allowing other internet users to request information from a router is usually a function for large scale databases or servers which host a variety of web pages and programs, but it can also be used by consumers to host their own sites or to create direct connections with other computers.

Firewalls

Routers can be susceptible to malicious activity because anyone on the internet can query them if they know the IP. In order to avoid this, average users have their routers protected by firewalls which filter traffic passing through the router and provides a layer of added security. Most standard commercial routers come with Firewalls already setup with no user configuration needed. And they prevent outside attackers from getting access into a router's network by requiring users on the router's network to verify incoming traffic.

Encryption

Encryption works with Firewalls to to make accessing information more secure. But unlike Firewalls, router encryption encrypts data that is sent from the device to the router so that it cannot be stolen along the local network. Although this is mainly a concern on public networks where there can be many users online at once, it is still important to have router passwords and allow encryption to deter people passing by. Again, like with Firewalls, encryption is often enabled by default and new routers have their default password usually located on the physical box.
