Browsers

Web browsers like “Google Chrome”, “Mozilla Firefox”, and “Internet Explorer” all are software applications which direct users to publicly available web pages or domains. Their goal is to make publicly available information easy to access. They work by interpreting URL's like www.google.com to query domains and get information from a resource. They interpret protocols to determine the type of request. Https: for example, is used for encrypted, secure web pages.

Cookies

Browsers also store user “cookies”, which are packets of browsing data that allows the website to retain browsing information such as site layouts, login information and are used to create recommended tabs on sites like Facebook and Amazon.These cookies are sometimes viewed as security risks because they can lead to stolen information if users visit untrusted sites, but in general they function to make browsing easier and more convenient.
