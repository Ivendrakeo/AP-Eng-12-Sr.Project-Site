Steps

The first thing I did was find a web template. Web programming is generally outside my scope, so I decided to find a web template that supported some of the aesthetics features found on this page like the top bar, side bar, and general page layout. Next I decided to setup the host and domain name. Although the school blocked my original attempt, I decided to get a domain that pointed to the same site as our original robotics scouting host. Code

This website is written in Html or Hyper Text Markup Language which is a "Programming" language for developing websites. Instead of using google sites or wordpress which cut out all the realistic steps in developing a website, I decided to use Html for the most customization and direct interaction with internet infrastructure. I use Filezilla for FTP upload to the website. I also used Putty to SSH and view the host logs (see pictures below).
